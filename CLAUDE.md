# Claude Code 設定リポジトリ

このリポジトリはClaude Codeの設定ファイルを管理します。

## ディレクトリ構成

| ディレクトリ | 内容 |
|-------------|------|
| `agents/` | カスタムエージェント定義 |
| `rules/` | コーディングルール・ガイドライン |
| `skills/` | カスタムスキル定義 |
| `settings.json` | グローバル設定 |

## 使用方法

このリポジトリの内容を `~/.claude/` にシンボリックリンクまたはコピーして使用します。

## 編集時の注意

- JSON/YAML/Markdown の構文エラーに注意
- 変更後は Claude Code を再起動して反映を確認
- シークレットや個人情報をコミットしない
